Errors in javascript:

  • Syntax error

    • Mistakes that occur while typing the code in javascript


  • Type error

    • Mistakes that occur while we print the code or output which is in different type than that we declare


  • Reference error

    • It Occurs when we print or try to access an undeclared variable in the code


  • Range error

    • The floating numbers can be added to the values of ranging in 0 to 100 only in javascript


Error handling in javascript:
  • try

    • If the logic is true then we will get the output of that code present in the try block


  • catch

    • If the logic is false then we will get the output of that code present in the catch block


  • finally

    • The final block Excutes only when one of the error handling funtions gets complied like if either try or catch gets satisfied then we get the codecompiled along the satisfied block
