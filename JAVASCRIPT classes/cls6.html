JAVASCRIPT CLASS 6

Different controls which are present in the js to perform actions like clicking,hovering,scrolling,etc.
in JavaScript the mouse functions can be given as:

  • click - click

  • dblclick - double click
  • contextmenu - right click
  • onmousedown - Scroll Up
  • onmouseup - Scroll down
  • onmouseover - mouse hover
  • onmouseleave - mouse leave
  • onmouseout - mouse out
  • onmousemove- mouse move
