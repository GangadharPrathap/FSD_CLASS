Arrays in javascript:
in order to insert the element in the array from the back we use push()
in order to remove the element in the array from the last we use pop()
in order to remove the element in the array from the first we use shift()
in order to insert the element in the array from the first we use unshift().
in order to remove a particular element in the array we use splice() {Here in the splice 'a' is the index position ; 'b' is the no of the items need to be removed after that particular index,'c,d,.....n' are the numbers that are going to be replaced}
in order to find the length we use the .length function
in order to sort the given array we use the .sort() function
in order to reverse the given array we use the .reverse() function
