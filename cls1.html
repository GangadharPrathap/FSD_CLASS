MEDIA QUERIES

Media queries help maintain layout responsiveness across screen sizes:

  • max-width: Applies up to a max screen size
  • min-width: Applies from a minimum screen size onward
Common device breakpoints:
  • Desktop: 1440px
  • Laptop: 1024px
  • Mobiles:
    • Android 13 and below: ~320px
    • Android 14+: 375px (portrait), 420px (landscape)
